# led-array-hplc-uvvis-detector
Development of a UV/VIS HPLC detector using components from a Linear 205 and an ADS1220 (24-bit) with a rotating LED array as light source

Linear 205 Components:
- Metal-Housing
- Optics (mechanical)
- Sensors (photodiode and AD650JN)

New components:
- ADS1220 (24-bit)
- Arduino Mega 2560 Pro Mini
- USB-C Communication
- Reed relay-controlled resistor network for fine-tuning
- UV/VIS-LED-Lamp with various wavelengths
- Newly developed housing modifications

Features:
- Improved measurement board (sensitivity, noise, shielding)
- New LED-Lamp-Array with broader spectral coverage
- Precise, low-noise 12-bit LED control board

Current developments:
- Lamp housing
- Signal Input-/Output-System (Events)
- Optimization, bug fixing and customization of the software
- System testing (robustness, reproducibility, detection limits, etc.)
